# My Blog

A modern, responsive static blog website created using HTML, CSS, and vanilla JavaScript. The design is clean, minimal, and professional, mimicking the aesthetics of platforms like Medium or Ghost themes.

## Features

- **Homepage**: Displays blog title, tagline, a grid of blog post cards, and a sidebar.
- **Single Blog Post Page**: Features a title, author, date, featured image, reading progress bar, table of contents, and formatted article content.
- **About Page**: Provides information about the blog’s mission.
- **Dark Mode Toggle**: Optional dark mode with localStorage persistence.
- **Responsive Layout**: Mobile-first design ensuring accessibility across devices.
- **Embedded JavaScript and CSS**: All styles and scripts are included directly in HTML files.
- **Accessibility**: Semantic HTML, ARIA labels, and appropriate alt text.

## Project Structure

```
/my-blog
    ├── index.html
    ├── post.html
    └── about.html
```

## Getting Started

1. Clone the repository or download the files.
2. Open `index.html` in your web browser to view the homepage.
3. Navigate to `post.html` and `about.html` for additional pages.

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Inter)
